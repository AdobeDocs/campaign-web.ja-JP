# 自動配信 {#automated-delivery}

>[!CONTEXTUALHELP]
>id="acw_orchestration_automated-delivery"
>title="配信アクティビティの自動化"
>abstract="**自動配信** アクティビティを使用すると、ワークフロー内で配信アクションを作成、設定、実行できます。 テンプレートから新しい配信を作成したり、個別に作成された既存の明示的な配信を選択したり、インバウンドトランジションで指定された配信を使用したりできます。 受信者とコンテンツを設定し、実行アクションを選択します。
